# Product Overview — D-Card

## Problem
Universities rely on physical ID cards that are:
- Easy to lose or damage
- Expensive to reissue
- Slow to revoke or update
- Inconvenient for guests and temporary access

## Solution
D-Card is a **digital-first university identity** that lives on a user’s mobile device and integrates with campus systems through secure validation.

## User Roles

### Students
- Campus access (libraries, labs, classrooms)
- Exam and attendance verification
- Wallet-based digital ID

### Staff
- Secure access to academic and administrative areas
- Role-based permissions

### Guests & Visitors
- Time-limited day passes
- Pre-approved access
- Automatic expiry

### Administrators
- Review and approve registrations
- Assign roles and access rules
- Monitor access logs

## Value Proposition
- Reduced operational cost
- Faster onboarding
- Improved security
- Better user experience
